Sobre a página

Esta página é o meu primeiro projeto em HTML, ele foi feito sem a ajuda de CSS ou JavaScript. Contudo, posteriormente planejo aprender essas linguagens para fazer uma nova versão do site melhorada. A ideia de fazer um site sobre filmes foi de um amigo meu (oi Hugo), que já fez um projeto similar na faculdade, e estou usando o site dele e do Shopping Boulevard de BH como base para criar o meu, não estou copiando uma linha sequer dessas duas páginas, apenas vendo como dispor os elementos de uma forma amigável.

Sobre o criador

Meu nome é João Monteiro, tenho 19 anos no momento. Acabo de passar na UFMG em Engenharia de Sistemas para o 2o Semestre, enquanto espero o curso começar estou fazendo cursos online para melhorar minhas habilidades. Após fazer dois cursos básicos de HTML, me desafiei a fazer um site completo apenas em HTML, mesmo sabendo que CSS e JavaScript são muito utilizados para este fim para aprender mais sobre essa linguagem.
